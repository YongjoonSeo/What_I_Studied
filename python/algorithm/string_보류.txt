문자열 - 패턴 매칭 알고리즘: 본문에서 특정한 문자열 찾는 것.
1. 고지식한 패턴 검색 알고리즘 (Brute-force)
2. KMP 알고리즘
3. 보이어-무어 알고리즘 
(a pattern matching algorithm) -> rithm

1. 본문 문자열을 처음부터 끝까지 차례대로 순회하면서
 패턴 내의 문자들과 일일이 비교하는 방식.
시간 복잡도 O(mn)

2. KMP알고리즘
불일치가 발생한 문자열 앞 부분에 대해 다시 비교하지 않고 매칭을 수행하는 방식.
	1. 패턴을 전처리해서 배열 next[M]을 구한다. 
	(next[M]: 불일치가 발생했을 경우 이동할 다음 위치)
시간 복잡도 O(n)

3. 보이어-무어 알고리즘
패턴의 오른쪽에서 왼쪽으로 비교. 
(보통 상황에서 앞부분 보다 끝부분에 가서 불일치가 일어날 확률이 높다는 것을 이용한 것)
패턴의 오른쪽 끝 문자가 불일치하고, 패턴 내에 이 문자가 없다면
이동거리는 패턴의 길이 만큼이 된다. 
패턴 내에 이 문자가 있다면 패턴 내에 일치하는 문자를 찾아 그 칸 수 만큼 점프한다.
(skip배열을 사용하여 구현할 수 있다)
시간 복잡도 일반적으로는 O(n)보다 적다. 텍스트를 모두 보지 않아도 검색 가능.
 (최악의 경우 O(mn))
